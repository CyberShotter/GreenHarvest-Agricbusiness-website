## Project Overview  
This is a modern, professional 5-page responsive website for **GreenHarvest**, a fictional but realistic agribusiness company that supplies seeds, fertilizers, training, equipment rental, and market linkage to farmers.

Built from scratch using pure HTML5 and CSS3 (no frameworks), the project demonstrates everything learned from the **FreeCodeCamp Responsive Web Design** certification and beyond.
